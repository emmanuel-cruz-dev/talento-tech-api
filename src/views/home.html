Proyecto Final - Backend Node.js

🔐 Autenticación requerida

Esta API requiere autenticación JWT para acceder a los endpoints.

📋 Endpoints disponibles:

  • POST /auth/login - Iniciar sesión
  • GET /api/products - Obtener productos (requiere token)
  • GET /api/products/:id - Obtener producto por ID (requiere token)
  • POST /api/products/create - Crear producto (requiere token)
  • DELETE /api/products/:id - Eliminar producto (requiere token)

👤 Usuarios de prueba:

  • Username: admin | Password: password123
  • Username: user1 | Password: password123

🛠 Cómo probar:

1. Usa Postman o curl para hacer login en /auth/login

2. Copia el token recibido

3. Agrégalo como Header: Authorization: Bearer tu_token
